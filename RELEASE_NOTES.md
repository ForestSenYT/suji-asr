# suji-asr v1.0 — 首个发布

中文讲课录音**批量转写** → SRT / VTT / JSON / MD。Windows 桌面应用,**全离线**,GPU/CPU 运行期自适应。目标:用聚合吞吐碾压实时转写工具(豆包等 ≈ 1× 实时)。

## 三种转写模式(GUI 顶部「转写模式」下拉切换)
| 模式 | 模型 | 速度(多文件批量) | 适用 |
|---|---|---|---|
| **准确度(默认)** | Qwen3-ASR-0.6B | ~6× 实时 | 人名 / 中英混术语最准 |
| **速度** | FireRedASR-AED fp16 | ~8–9× 实时 | 量大、对人名不挑剔 |
| **词级字幕** | FireRedASR2-CTC | ~4–5× 实时 | 需要精确到词的字幕时间戳 |

> 实测于 RTX 2080 + Ryzen 5800X;部署机(3070 Ti + 更新 i7)预计更快。

## 本版亮点
- **三模型一体**,GUI 一键切换,默认走 **Qwen3** 准确度最高(中英混人名也能转对,如「奥斯特罗姆 / OLIVER WILLIAMSON」)。
- **修复 fp16 模型转写乱序 bug**:注意力模型不产逐 token 时间戳,旧版用了不稳定排序导致字序错乱;现改稳定排序,逐字节正确。
- **多文件批量数据并行**:Qwen3 批量聚合吞吐 **+60%**(双 CPU recognizer,K=2)。
- **运行期自适应**:检测到 N 卡 + CUDA 运行时 → 自动用 GPU,否则 CPU,零配置、崩溃安全。
- 每文件独立进度条 + 实时倍速/ETA、彩色时间戳日志、拖拽、右键打开输出、取消、**断点续跑**、UTF-8 中文文件名全程不乱码。

## 安装
- 下载 **`suji-asr-setup-1.0.exe`**(~4.5GB,all-in-one:4 个模型 + CUDA 12 运行时 + Qt + ffmpeg + VC++ 运行时,**干净机即装即用,无需任何额外下载或环境变量**)。
- 系统:Windows 10/11 x64;可选 NVIDIA GPU(CUDA 12)加速;约 7GB 安装空间。

## 已知事项 / 限制
- Qwen3 是自回归 LLM 解码器,单流 ~4.5×、批量 ~6× 已是消费级硬件的现实上限(GPU 对其 int8 提速有限;CPU 是主力)。想更快需 fp16 导出(收益有限)或更强 CPU。
- 字幕时间戳:Qwen3 / AED 模式是「每语音段一条」;需要词级请切「词级字幕(CTC)」模式。
- **许可证**:分发前请自行核对 Qwen3(ModelScope 社区 ONNX 导出)、FireRedASR、CUDA redistributable、ffmpeg-lgpl、VC++ 运行时的再分发条款。
